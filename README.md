# Simple Exchange Rates

An example of a simple exchange rates web apps built in React Redux based on Create React App.

### Access Locally

```
$ git clone https://github.com/yasintze/simple-exchange-rates.git
$ cd simple-exchange-rates
$ yarn install
$ yarn run start
```
